# Next.js (App Router), Panda CSS and Storybook example project

- [Next.js] v14 (App Router)
- [Panda CSS] v0
- [Storybook] v7

Demo  
<https://cieloazul310.github.io/next-panda-example/>

## Project structure

```txt
.
├── src
│   ├── app // Next.js App Router 
│   ├── components
│   ├── mdx-components.tsx
│   ├── stories // Storybook stories
│   ├── styles
│   ├── theme // for Panda config
│   └── utils
├── next.config.mjs
├── package.json
├── panda.config.ts
├── postcss.config.cjs
├── public
└── tsconfig.json
```
