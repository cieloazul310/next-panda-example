## Project structure

```txt
.
├── .storybook
├── content
│   └── post // MDX posts
├── public
├── src
│   ├── app // Next.js App Router 
│   ├── components
│   ├── lib
│   ├── stories // Storybook stories
│   ├── styles
│   ├── theme // for Panda config
│   ├── utils
│   └── mdx-components.tsx
├── next.config.mjs
├── package.json
├── panda.config.ts
├── park-ui.json
├── postcss.config.cjs
└── tsconfig.json
```
